A lambda function is a short and anonymous function that takes one or more parameters and contains a single expression. and these functions are pure always. (minimum one parameter and maximum 1 expression) A JavaScript Lambda Function is an anonymous function written without a name or a function definition. Lambda functions are arrow functions Lambda functions are a concise and convenient way to write small functions for a specific purpose, such as transforming an array of data or filtering a list of items. Mostly they are used in map , filter, reduce ,some , find etc array method
